# 📌 Resumo Técnico

A manutenção preditiva é uma estratégia baseada em dados que permite identificar falhas antes que ocorram.

No contexto offshore, essa abordagem é essencial devido ao alto custo de parada e risco operacional.

A Inteligência Artificial pode analisar padrões de comportamento em sistemas elétricos, como variações de temperatura, vibração e corrente elétrica.
